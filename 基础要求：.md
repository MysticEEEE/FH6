# 基础要求

本文档用于记录运行 FH6Auto 前建议完成的车辆、蓝图、游戏设置和账号资源准备。

## 车辆准备

建议使用以下车辆作为跑图车辆：

- 车辆：Subaru Impreza 22B-STi Version
- 推荐调校：S2
- 车辆调校代码：`117629113`

请提前完成以下准备：

- 将目标车辆加入车库，并确保可正常选择。
- 保持车辆可用于当前蓝图和跑图流程。
- 如果车库内还有其他 Subaru 车辆，建议先手动驾驶或进入查看一次，尽量清除无关车辆上的“全新”标识，减少自动选车时的干扰。

## 蓝图准备

- 蓝图代码参考：`659086805`

蓝图代码可根据实际游戏版本和个人需求替换。如果游戏更新导致流程或识别位置变化，建议重新确认蓝图是否仍可正常进入、结算和重复运行。

## 游戏设置

推荐使用以下设置，以提高图像识别和自动操作稳定性：

- 显示模式：窗口模式或无边框窗口模式。
- 游戏语言：简体中文。
- 输入法：英文键盘。
- HUD：关闭可能遮挡界面的额外提示。
- 换挡方式：手动挡。
- 辅助驾驶：开启自动转向。
- 分辨率和缩放：尽量保持固定，不要在运行过程中切换。

运行过程中请尽量避免频繁切换窗口、拖动游戏窗口或改变画质设置，否则可能影响截图识别结果。

## 资源准备

请确保账号内保留一定数量的 CR，避免自动买车时因余额不足导致流程中断。

如果要长时间循环运行，建议在开始前确认：

- 车辆库存空间充足。
- 技能点数量足够当前计划。
- 网络连接稳定。
- 游戏已正常进入可操作状态。

## AI 辅助准备

如果需要使用 `AI辅助`：

- 下载 AI 版 Release，或从 Release 附件中获取预训练模型。
- 将模型放到 `models/fh6_car_select_yolo.pt`。
- 从源码运行时，需要安装 `requirements-ai.txt`。

当前预训练模型主要基于 1080p 数据训练。非 1080p 分辨率可以尝试使用，但如果识别不稳定，建议补充对应分辨率截图并重新训练。

## 运行建议

- 第一次使用或更新模板后，建议先用较小次数测试完整流程。
- 超级抽奖选车依赖 `全新标签 -> B600 等级 -> 目标车辆卡片` 的三段验证。
- 如果识别失败，可优先检查 `images` 目录中的相关模板是否需要重新截图替换。
- 如果遇到识别异常、卡菜单或流程跳转失败，先停止任务，再手动恢复到稳定界面后重新开始。

更多功能说明和打包方式请参考 `README.md`。
